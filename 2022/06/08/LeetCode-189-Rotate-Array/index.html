LeetCode 189.Rotate Array

Posted on 2022-06-08 In algorithm Symbols count in article: 838 Reading time ≈ 1 mins.

This article is recording How to solve the LeetCode question Rotate Array.

Question

Given an array, rotate the array to the right by k steps, where k is non-negative.

Constraints

  • 1 <= nums.length <= 105
  • -231 <= nums[i] <= 231 - 1
  • 0 <= k <= 105
  1. assigned k is k % nums length. This action is because protects when k is greater than nums length.
  2. create method to reverse nums.
  3. first reverse means reverse “—–>–>” we can get “<–<—–”.
  4. second reverse “<–” we can get “–><—–”.
  5. third reverse “<—–” we can get “–>—–>”.
